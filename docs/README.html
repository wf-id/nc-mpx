Monkeypox Cases in North Carolina

This repository saves a copy of the posted reported monkeypox (MPX) cases for NC as reported by NCDHHS on their website.

Key Functions

  • scrape.R saves a local snapshot of the aggregated cases as a json file in the data directory

Key Output

  • Cumulative and new daily cases will be available in the mpx.csv file. The date reflects the last available update.

Additionally a graph of these data are hosted at this website.
